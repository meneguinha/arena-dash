# Grêmio Arena Dashboard

Dashboard interativo para análise financeira e de bilhetagem da Arena do Grêmio.

## Funcionalidades
- Carregamento automático de dados via `dados_gremio_processados.xlsx`.
- Filtros dinâmicos por Ano e Visitante.
- Gráficos interativos (ApexCharts):
  - Top 10 Setores por Arrecadação.
  - Bilhetes Vendidos por Setor (Donut).
  - Evolução da Arrecadação por Jogo (Timeline).
- Tabela de dados detalhada com ordenação dinâmica.
- Design Responsivo e Premium.

## Como usar
Basta abrir o `index.html` em qualquer navegador. No GitHub Pages, o dashboard carregará automaticamente os dados do arquivo Excel presente na raiz.
